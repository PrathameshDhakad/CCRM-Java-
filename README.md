# CCRM-Java-
Campus Course &amp; Records Manager (CCRM) is a Java SE console application for managing students, courses, enrollments, grades, and transcripts. It supports CSV import/export, backups, GPA reports, and a menu-driven workflow, while demonstrating OOP, Java Streams, NIO.2, exceptions, and designÂ patterns.
